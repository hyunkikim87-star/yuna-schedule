# Yuna Schedule — 타임그리드 UI 개편 스펙

## Overview

기존 카드뷰를 구글 캘린더 스타일의 모바일 타임그리드로 전면 개편.
토/일 2컬럼 + 해당 주에 공휴일이 있으면 해당 요일 컬럼 추가.

---

## 변경 사항 요약

| 항목 | 기존 | 변경 후 |
|------|------|---------|
| 레이아웃 | 주차 드롭다운 + 카드 2컬럼 | 타임그리드 (시간 세로축 × 요일 가로축) |
| 시간 데이터 | 없음 | start_time, end_time 추가 |
| 주 이동 | 드롭다운만 | 년/월 드롭다운 + ◀▶ 버튼 |
| 수업 추가 | + 버튼 (각 컬럼 하단) | FAB(+) 버튼 우하단 고정 |
| 수업 수정 | 카드 탭 | 이벤트 블록 탭 |
| 공휴일 | 없음 | 해당 주 공휴일 컬럼 자동 추가 |

---

## Data Model

### DB 마이그레이션

```sql
ALTER TABLE schedules
  ADD COLUMN start_time VARCHAR(5),  -- 'HH:MM', e.g. '10:00'
  ADD COLUMN end_time   VARCHAR(5);  -- 'HH:MM', e.g. '11:30'
```

기존 데이터 호환 유지 (nullable). 시간 미입력 시 그리드 상단 "시간 미정" 영역에 표시.

### 주차 계산

기존 `week` 필드 유지 (해당 월의 n번째 토요일 = n주차).  
공휴일은 `date` (YYYY-MM-DD)로 직접 매핑해서 해당 주차 계산.

---

## API 변경

기존 API 유지, 응답 필드에 `start_time`, `end_time` 추가.

| Method | Path | 변경 내용 |
|--------|------|-----------|
| GET | `/api/schedules?year=&month=&week=` | 응답에 start_time, end_time 포함 |
| POST | `/api/schedules` | body에 start_time, end_time 추가 |
| PUT | `/api/schedules/:id` | body에 start_time, end_time 추가 |
| GET | `/api/holidays?year=&month=&week=` | 해당 주차 공휴일 목록 반환 (신규) |

## UI 설계

### 상단 앱바
- 앱 타이틀 "유나 수업 일정"
- 년 드롭다운 + 월 드롭다운 + ◀ ▶ 주 이동 버튼
- 현재 주차 날짜 범위 표시 (예: "3주차 · 6/14(토) ~ 6/15(일)")

### 타임그리드
- **시간 범위**: 오전 9시 ~ 오후 6시 (1시간 단위 행, 56px/시간)
- **컬럼 구성**: [공휴일(있을 때만)] + 토요일 + 일요일
- **공휴일 컬럼**: 헤더에 날짜·요일·공휴일명 표시, 빨간색 강조
- **이벤트 블록**: start_time 기준 top 절대 위치, 시간 길이 기준 height
  - 수업명(굵게), 시간, 장소 표시
  - 5가지 색상 (수업별 자동 배정 또는 사용자 선택)
  - 탭하면 수정 팝업
- **시간 미정 수업**: 그리드 상단 별도 영역 ("시간 미정" 섹션)

### FAB (+) 버튼
- 우하단 고정
- 탭하면 수업 추가 팝업

### 수업 추가/수정 팝업
- 입력 필드: 수업명*, 요일(토/일/공휴일)*, 시작 시간, 종료 시간, 장소, 준비물, 메모
- 시간은 30분 단위 드롭다운 (선택 안 함 = 시간 미정)
- 저장 / 취소 버튼
- 수정 모드: 삭제 버튼 추가

## 공휴일 데이터

`src/holidays.js`에 하드코딩. 대상 연도: 2025~2030.

**고정 공휴일**: 신정(1/1), 삼일절(3/1), 어린이날(5/5), 현충일(6/6), 광복절(8/15), 개천절(10/3), 한글날(10/9), 성탄절(12/25)

**음력 공휴일** (연도별 양력 날짜 사전 계산 후 하드코딩):
- 설날 연휴 (음력 1/1 전후)
- 부처님오신날 (음력 4/8)
- 추석 연휴 (음력 8/15 전후)
- 대체공휴일 포함

**로직**: 해당 주차(토~일)에 공휴일이 포함되면 `/api/holidays` 응답에 포함. 프론트엔드에서 컬럼 동적 추가.

## 컴포넌트 구조

```
App
├── AppBar (년/월 드롭다운, 주 이동 버튼)
├── WeekView (타임그리드 전체)
│   ├── DayHeaders (요일 헤더 행)
│   ├── TimeGrid (시간축 + 이벤트 배치)
│   │   ├── TimeColumn (9시~18시 레이블)
│   │   └── DayColumn × N (공휴일/토/일)
│   │       └── EventBlock (수업 블록)
│   └── AllDayRow (시간 미정 수업 영역, 있을 때만)
└── EventModal (추가/수정 공통 팝업)
```
